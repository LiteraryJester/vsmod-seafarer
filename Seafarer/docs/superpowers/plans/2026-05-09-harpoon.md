# Harpoon Weapon Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a thrown harpoon weapon to the Seafarer mod. Functions as a javelin (charge-aim, throw, damage) with a material-scaled chance to tether the struck entity to the thrower via a physical rope. Rope breaks if stretched too far, leaving the harpoon embedded in a "no-rope" state that needs re-roping.

**Architecture:**
- Item assets use the base-game `ItemSpear` class for throw mechanics (no custom item class needed). Two distinct items per metal: `harpoon-{metal}` (with rope) and `harpoon-norope-{metal}` (without). A custom projectile entity class `EntityProjectileHarpoon` extends `EntityProjectile` and runs the tether-on-impact logic. A new server-side `RopeTetherTracker` mod system polls active tethers and breaks the rope when stretched past tolerance, swapping the embedded harpoon's drop ItemStack from the rope variant to the no-rope variant.

**Tech Stack:** C# (.NET 10), Vintage Story 1.21+ modding API. Uses base-game `ItemSpear`, `EntityProjectile`, `ClothSystem`, `ClothManager`, `EntityBehaviorRopeTieable`. No automated test infrastructure for in-game weapon code; verification is build-clean + asset-validator + manual play.

```csharp
using Vintagestory.API.Common;
using Vintagestory.API.Common.Entities;
using Vintagestory.API.Datastructures;
using Vintagestory.API.MathTools;
using Vintagestory.GameContent;
using Seafarer.Systems;

namespace Seafarer
{
    public class EntityProjectileHarpoon : EntityProjectile
    {
        protected override void ImpactOnEntity(Entity entity)
        {
            base.ImpactOnEntity(entity);

            if (World.Side != EnumAppSide.Server) return;
            if (entity == null || !entity.Alive) return;
            if (FiredBy == null || !FiredBy.Alive) return;
            if (ProjectileStack?.Collectible == null) return;

            JsonObject attrs = ProjectileStack.Collectible.Attributes;
            if (attrs == null) return;

            bool isRoped = attrs["isRoped"].AsBool(false);
            if (!isRoped) return;

            if (entity.Properties?.Attributes?["harpoonImmune"].AsBool(false) == true) return;

            float tetherChance = attrs["tetherChance"].AsFloat(0f);
            if (tetherChance <= 0f) return;
            if (World.Rand.NextDouble() > tetherChance) return;

            TryTether(entity, attrs);
        }

        private void TryTether(Entity target, JsonObject attrs)
        {
            var ropeTieable = target.GetBehavior<EntityBehaviorRopeTieable>();
            if (ropeTieable == null)
            {
                ropeTieable = new EntityBehaviorRopeTieable(target);
                ropeTieable.Initialize(target.Properties, null);
                target.AddBehavior(ropeTieable);
            }
            if (!ropeTieable.CanAttach()) return;

            var cm = Api.ModLoader.GetModSystem<ClothManager>();
            if (cm == null) return;

            Vec3d throwerPos = FiredBy.SidedPos.XYZ.Add(0, FiredBy.LocalEyePos.Y - 0.3f, 0);
            Vec3d targetPos = target.SidedPos.XYZ.Add(0, 0.5f, 0);

            ClothSystem rope = ClothSystem.CreateRope(Api, cm, throwerPos, targetPos, null);

            Vec3d throwerOffset = new Vec3d(0, FiredBy.LocalEyePos.Y - 0.3f, 0);
            rope.FirstPoint.PinTo(FiredBy, throwerOffset.ToVec3f());

            ropeTieable.Attach(rope, rope.LastPoint);

            cm.RegisterCloth(rope);

            float nominalLength = attrs["ropeMaxLengthBlocks"].AsFloat(16f);
            float breakLength = attrs["ropeBreakLengthBlocks"].AsFloat(24f);

            var tracker = Api.ModLoader.GetModSystem<RopeTetherTracker>();
            if (tracker != null)
            {
                tracker.RegisterTether(new TetherRecord
                {
                    ClothId = rope.ClothId,
                    HarpoonEntityId = EntityId,
                    ThrowerEntityId = FiredBy.EntityId,
                    TargetEntityId = target.EntityId,
                    NominalLength = nominalLength,
                    BreakLength = breakLength
                });
            }
        }
    }
}
```

Create `Seafarer/Seafarer/Systems/RopeTetherTracker.cs`:

```csharp
using System.Collections.Generic;
using Vintagestory.API.Common;
using Vintagestory.API.MathTools;
using Vintagestory.API.Server;
using Vintagestory.GameContent;

namespace Seafarer.Systems
{
    public class TetherRecord
    {
        public int ClothId;
        public long HarpoonEntityId;
        public long ThrowerEntityId;
        public long TargetEntityId;
        public float NominalLength;
        public float BreakLength;
    }

    public class RopeTetherTracker : ModSystem
    {
        private ICoreServerAPI sapi;
        private ClothManager cm;
        private readonly List<TetherRecord> tethers = new();
        private long tickListenerId;

        public override bool ShouldLoad(EnumAppSide side) => side == EnumAppSide.Server;

        public override void StartServerSide(ICoreServerAPI api)
        {
            sapi = api;
            api.Event.SaveGameLoaded += OnSaveGameLoaded;
        }

        private void OnSaveGameLoaded()
        {
            cm = sapi.ModLoader.GetModSystem<ClothManager>();
            tickListenerId = sapi.Event.RegisterGameTickListener(OnSlowTick, 1000);
        }

        public override void Dispose()
        {
            if (sapi != null && tickListenerId != 0)
            {
                sapi.Event.UnregisterGameTickListener(tickListenerId);
                tickListenerId = 0;
            }
        }

        public void RegisterTether(TetherRecord record)
        {
            tethers.Add(record);
        }

        private void OnSlowTick(float dt)
        {
            if (cm == null) return;
            for (int i = tethers.Count - 1; i >= 0; i--)
            {
                var t = tethers[i];
                if (ShouldDrop(t, out bool shouldBreak))
                {
                    if (shouldBreak) BreakTether(t);
                    tethers.RemoveAt(i);
                }
            }
        }

        private bool ShouldDrop(TetherRecord t, out bool shouldBreak)
        {
            shouldBreak = false;

            ClothSystem rope = cm.GetClothSystem(t.ClothId);
            if (rope == null) return true;

            Entity thrower = sapi.World.GetEntityById(t.ThrowerEntityId);
            Entity target = sapi.World.GetEntityById(t.TargetEntityId);
            if (thrower == null || !thrower.Alive || target == null || !target.Alive)
            {
                shouldBreak = true;
                return true;
            }

            Vec3d a = rope.FirstPoint.Pos;
            Vec3d b = rope.LastPoint.Pos;
            double dist = a.DistanceTo(b);
            if (dist > t.BreakLength)
            {
                shouldBreak = true;
                return true;
            }

            return false;
        }

        private void BreakTether(TetherRecord t)
        {
            ClothSystem rope = cm.GetClothSystem(t.ClothId);
            if (rope != null)
            {
                rope.WalkPoints(p => p.UnPin());
                cm.UnregisterCloth(t.ClothId);
            }

            Entity target = sapi.World.GetEntityById(t.TargetEntityId);
            target?.GetBehavior<EntityBehaviorRopeTieable>()?.Detach(rope);

            Entity harpoon = sapi.World.GetEntityById(t.HarpoonEntityId);
            if (harpoon is EntityProjectile proj && proj.ProjectileStack?.Collectible != null)
            {
                string oldCode = proj.ProjectileStack.Collectible.Code.Path;
                if (oldCode.StartsWith("harpoon-") && !oldCode.StartsWith("harpoon-norope-"))
                {
                    string metal = oldCode.Substring("harpoon-".Length);
                    var noRopeItem = sapi.World.GetItem(new AssetLocation("seafarer", "harpoon-norope-" + metal));
                    if (noRopeItem != null)
                    {
                        proj.ProjectileStack = new ItemStack(noRopeItem, 1);
                    }
                }
            }
        }
    }
}
```

## Notes for the implementer

- **Class registration is server-driven for entities.** The `api.RegisterEntity("EntityProjectileHarpoon", typeof(EntityProjectileHarpoon))` call must be in `Start()` (early), not `StartServerSide()`. This matches the existing `EntityProjectileBarbed` pattern.
- **`ItemSpear` from base game is used directly** — no `ItemHarpoon` C# class. The harpoon items reference `class: "ItemSpear"` in JSON, and `spearEntityCodeByType` routes to our custom projectile entity. This avoids reimplementing the throw lifecycle.
- **`ClothSystem.CreateRope` requires `ICoreAPI`, `ClothManager`, two `Vec3d` positions**, and an optional `clothSectionModel` (pass `null` for default rope). Returns the new system; remember to `cm.RegisterCloth(rope)` after pinning both ends.
- **`EntityBehaviorRopeTieable.Attach(rope, point)` handles pinning + cloth-id tracking on the target entity.** Don't manually call `point.PinTo(target, ...)` for the target end — `Attach` does it correctly and updates `target.WatchedAttributes["clothIds"]` so the cloth survives a chunk reload.
- **For the thrower end, you DO call `point.PinTo(throwerEntity, offset)` directly** because the thrower (a player) doesn't need cloth-id persistence on their behavior.
- **`*ByType` resolves at JSON load time.** `tetherChanceByType` becomes `tetherChance` for each variant. The projectile reads `attrs["tetherChance"]` directly, not the keyed dictionary.
- **The `ProjectileStack` field on `EntityProjectile` is settable.** Task 7's break-rope step swaps it from the rope variant to the no-rope variant by replacing the ItemStack reference. The base class drops `ProjectileStack` on retrieval, so the player gets whatever code is current at the moment of retrieval.
- **`api.ModLoader.GetModSystem<ClothManager>()` returns null if cloth physics aren't loaded.** This shouldn't happen in vanilla VS but the null-guard is cheap insurance.
- **`RopeTetherTracker.OnSlowTick` runs every 1000ms.** Keep work minimal: just distance checks. Heavy work belongs elsewhere.
- **The first task's smithing pattern is a guess** at a reasonable harpoon-head shape on the anvil. Adjust if the in-game anvil rejects it (the validator can't catch invalid smithing patterns).
